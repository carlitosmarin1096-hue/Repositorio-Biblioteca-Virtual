# 📚 Biblioteca Virtual - Repositorio de Tesis

Sistema web desarrollado en PHP y MySQL para la gestión de documentos académicos.

## 🚀 Funcionalidades
- Login de usuarios
- Roles (administrador, docente, estudiante)
- Subida de documentos PDF
- Búsqueda de archivos
- Descarga de documentos
- Panel administrativo (dashboard)

## 🛠️ Tecnologías
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript

## 📂 Estructura
- dashboard.php
- login.php
- subir.php
- buscar.php
- config/conexion.php
- uploads/
